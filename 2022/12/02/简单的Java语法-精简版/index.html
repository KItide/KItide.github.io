    简单的Java语法

    从简单的java程序入手

    先说注意事项:

    *java区分大小写

    *java的类（class）名必须和文件名保持一致

    *java中使用 . 来调用某个对象的方法

    *java中使用 ; 作为每个语句的结束符

    对上面的代码一行行分析：
    1.public class HelloWorld

    public : java修饰符的一种,这里对类HelloWorld进行修饰，顾名思义，被public修饰的类是“公共的，谁都可以访问的”，现在不理解不要紧，只需要知道目前的新手期用它就对了（修饰符知识点涉及涉及访问权限，之后会详细讲解）

    class : 类的定义标识，定义某个类必须要加上它（理解为固定格式就行）

    2.public static void main(String[] args) :

    public : 同上，但这里修饰的是main()方法，main()方法必须用public修饰

    static : 这个现在不需要过多了解，新手期可以把它当成写main()方法的固定格式

    void : 无返回值类型，这里特别讲一下，方法调用完成后通常会返回一个值（值的类型由自己写的代码决定），可以通过这个值用来反馈调用情况或方法得出的结果，而void则是没有返回值

    main(String[] args) : 方法的主体格式为 “ 返回值类型 方法名 （参数定义）”，方法名一般由你自己决定，但main()方法是java已经默认定义好的主方法，它是程序的入口，而括号里的String[] args 是main()方法的参数，这里把它当成一个main()的固定格式就好了

    3.System.out.println("Hello,World");

    System.out : 它的情况有点特殊，涉及被某个学长删改掉的Java底层介绍，这里不对它进行介绍，有兴趣的自行百度了解

    .println() : 这个就是定义在System类中的println()方法，用 . 来调用，括号里的是要输出（或打印）到屏幕上的内容，括号里可以放入你定义的变量，也可以直接放入字符串如"Hello,World"

    数据类型：

    整型：

    int 4 short 2 long 8 byte 1

    有上面写的四种，日常常用的就只有int，其他的需要用到再讲

    浮点类型：（简单地理解为小数）

    float（小数点后位数少，精度没那么高，但日常够用）

    double（小数点后位数多，精度高，一般不常用）

    float 类型的数值有一个后缀 F 或 f (例如，3.14F。) 没有后缀 F 的浮点数值（如 3.14 ) 默 认为 double 类型。当然，也可以在浮点数值后面添加后缀 D 或 d (例如，3.14D) 。

    char类型：

    先简单理解为用单引号引用起来的字符

    boolean类型 ：

    俗称布尔类型，只有两个值true（真），false（假） ，一般用于两种状态的判断

    变量:

    变量名字自己定义，但不能有中文
    变量名必须是一个以字母或下划线_开头,且不能加入键盘上的奇怪符号.
    可以同时定义 eg：int i , j ;
    定义后必须要初始化（给变量一个确定的值） eg：i=0；
