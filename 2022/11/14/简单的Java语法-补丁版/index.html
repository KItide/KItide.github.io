    3.System.out.println("Hello,World");

    这是Java中很常见的输出语句。由 System.out 提供的输出方法，可以在输出信息后追加一个换行，除此以外 System.out 提供的输出方法还有：

    System.out.print()输出之后不追加换行；

    一种简单的数据结构——数组

    关于数组

    数组是一种数据结构， 用来存储同一类型值的集合。通过一个整型下标可以访问数组中 的每一个值。例 如， 如果 a 是一个整型数组， a[i] 就是数组中下标为 i 的整数。

    是不是看得很迷糊（滑稽）？ 来看通俗一点的解释吧：

    1. 如果给你一堆同样类型的数据让你储存起来，就比如班上40个人，统计分数，(假设大家考的都是整数分)，总不可能你自己在程序的源代码里一个个地声明、初始化40个整型变量吧(不会真的有人想这么做吧)，这个时候，我们就可以使用数组简单的声明一下

      这样，可以储存40个整型数据的位置就开辟出来了，剩下的就只用初始化就行了。

    2. 那么我们把40个数据填充进去后，如果想更改原来某个分数（或者查看某个分数）时，该怎么办呢？不要担心，数组设置了下标这个东西来对数组的某个位置进行“标记”，通过对这个位置的引用，就可以实现对 原本放入这个位置的元素 的访问和更改操作。

      注意:
      • 数组的下标从 数字0开始而不是数字1，例如，长度为40个元素的数组下标范围是 0 ~ 39，而不是0 ~ 40，如果此时下标使用了40，会报错（数组越界）
      • 在创建了数组后，只有给数组初始化之后，该数组才允许被访问
      • boolean 数组的元素会初始化为 false 对象数组的元素则初始化为一个特殊值 null（这里暂时不理解也不要紧，后面会讲到）
