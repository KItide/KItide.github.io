发表于 2021-05-10 更新于 2022-04-09 分类于 教程

C语言的四大数据类型

（一）基本类型（算数类型）

1.整数类型（整型）：short、int、long、long long

另外可用修饰符unsigned修饰这几种类型，分别有：

unsigned int、unsigned short、unsigned long、unsigned long long

2.浮点数类型（实型）：float、double、long double
3.字符型：char

另外可用修饰符signed、unsigned修饰：unsigned char、signed char

（二）枚举类型——规定有限个数据值作为此数据类型可以表示那些数据

1.整数类型

如果在定义枚举的元素时没有给其初始化，它的值会被自动初始化为其前一元素加 1。

而第一个枚举元素若没有被初始化，默认值就为整型（int）的 0

2.浮点数类型(同上，不过里面的枚举元素都是浮点数类型)

（三）Void类型（无值类型）

1.函数返回值类型

即 以void类型为返回值的函数不返回任何结果或数据

2.函数参数类型

例如int Fun(void) 就表示一个返回值为int数据、没有参数的函数Fun()

3.指针指向地址

void * p

好像是定义指针p后没有为其指定具体指向的数据类型，可以把该类型下的指针p强制转化成 可以指向任何数据类型的指针

（四）派生类型

1.构造类型——由程序员自己根据需要并结合基本类型而构造声明出的数据类型
数组——可以连续存储多个指定类型的数据的一个集合？
结构体——用于自定义一些符合实际需求的非基本的数据类型
共用体——特殊用处的类结构体
2.指针类型

指针指向的数据的类型 * 指针名 ，例如：int * p 、 char * pt
