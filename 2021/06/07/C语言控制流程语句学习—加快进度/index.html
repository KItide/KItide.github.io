    2. 带有else的if语句

      • if··· else

        一个 if 后可跟零个或一个 else，当if后的判断表达式不成立（为假）时，直接执行else后的语句

        同理，若大括号里的代码只有一行，则大括号可以省略不写

      • if··· else if··· else
        • 一个 if 后可跟零个或多个 else if，else if 必须在 else 之前。
        • 一旦if或者某个 else if 匹配成功，其他的 else if 或 else 将会被直接忽略跳过

      控制语句 描述
      break 语句 立即终止包裹着该break的最内层的循环或 switch 语句，程序流将继续执行紧接着循环或 switch 的下一条语句。
      continue 语句 告诉一个包裹着该continue的最内层的循环体立刻停止本次循环迭代，重新开始下次循环迭代。
      goto 语句 将控制转移到被标记的语句。但是不建议在程序中使用 goto 语句（狗都不用）
